# lol_mw_icon
script to upload lol icons to mw

## build
install dependencies
```
npm install
```
and then build it
```
npm run build
```

## how2use
entry points:
 - ```/dist/indexes.js``` - update indexes
 - ```/dist/icon.js``` - check + update icons
